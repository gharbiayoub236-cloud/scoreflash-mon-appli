# ⚽ ScoreFlash

ScoreFlash est une application mobile développée avec Flutter permettant de suivre les scores sportifs en direct et les matchs à venir.

---

## 🚀 Fonctionnalités

- 📊 Affichage des matchs en direct et à venir
- ⚽ Football (Serie A, etc.)
- 🏀 Basket (NBA)
- 🤾 Handball
- 🏐 VolleyBall
- 🔄 Filtrage par sport
- 🔐 Interface de connexion(sans BD)
- 📱 Navigation avec Drawer
---
## 🛠️ Technologies utilisées
- Flutter
- Dart
- REST API (TheSportsDB)
- HTTP package
---
## 📡 API utilisée
Les données des matchs sont récupérées depuis :
https://www.thesportsdb.com/
